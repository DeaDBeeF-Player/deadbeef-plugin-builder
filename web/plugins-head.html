Plugins

Installation

GNU/Linux

Download and extract .zip into ~/.local/lib/deadbeef/

NOTE: You may need to create this folder yourself, if it's not there

In the portable builds, it’s possible to unzip directly into deadbeef folder.

Windows

Download and extract .zip into %APPDATA%/deadbeef/

NOTE: You may need to create this folder yourself, if it's not there

In the portable builds, it’s possible to unzip directly into deadbeef folder.

macOS

Download and unpack zip, then copy *.dylib files into ~/Library/Application Support/Deadbeef/Plugins
